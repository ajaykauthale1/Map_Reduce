Packages:
Each Model has its own package as follows

seq - SEQUENTIAL
nolock - NO_LOCK
coarselock - COARSE_LOCK
finelock - FINE_LOCK
nosharing - NO_SHARING

Each package has 2 programs each: with Fibonacci (*fib.java) and without Fibonacci

Addidional Packages:
Addidional packages are as follows

util ->
contains 2 classes 
LoaderRoutine - to load the input file
ProcessUtility - to process the input data into meaningful one

resources ->
contains output for each program

model ->
Contais StationModel class for storing the weather data by station.

How to run?
Run program as java application
